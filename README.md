# 📋 people-list-api

👩‍💻 Projeto baseado no [desafio da e-Core](http://github.com/fm1randa/desafio-ecore) para estudo do TypeScript

🎲 A lista pode ser visualizada e manipulada pelo [site](http://github.com/fm1randa/people-list)!

## Instalação

### Instale o [Node](https://nodejs.org/en/download/) e o [Git](https://git-scm.com/downloads)

Clone o projeto:

```console
$ git clone https://github.com/fm1randa/people-list-api
```

No diretório do projeto rode o comando no terminal para instalar as dependências:

```console
$ npm install
```

ou

```console
$ yarn
```

<br>

## Execução

```console
$ npm run dev
```

ou

```console
$ yarn dev
```

### A API rodará no [http://localhost:3333](http://localhost:3333).
